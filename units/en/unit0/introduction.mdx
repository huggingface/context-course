# Welcome to The Context Course

Welcome to **The Context Course** — the essential guide to mastering context engineering for AI code agents.

## The Key Skill for Code Agents

Whether you're using Claude Code, Codex, or OpenCode, the same fundamental truth applies: **your agent is only as good as the context you give it**.

Code agents are great, but they're only as good as the context you give them. This is sometimes called **context engineering**: the art and science of structuring knowledge so that agents can access exactly what they need, when they need it.

This course teaches you how to refine your code agents' context so they can do their best work. You'll learn to build portable skills, integrate specialized tools through MCPs (Model Context Protocol), create custom plugins, spawn sub-agents for complex tasks, and optimize execution for maximum efficiency. 

These are the techniques that separate code agent blunders from success.

## How to Navigate This Course

### Recommended Pace

We recommend one unit per week, roughly 2–3 hours each, with hands-on practice throughout. Context engineering is not a quick skill — it requires practice and iteration. Rushing through units won't serve you well.

### Learning Format

Each unit combines conceptual lessons (the "why" and "how") with practical code examples you can run immediately, a hands-on project where you build something real, and a quiz to test your understanding.

### Customizing Your Path

While we recommend following units in order, you can customize based on your needs:
- **Just want skills?** Start with Unit 1, revisit MCPs when needed
- **Building a plugin for your team?** Start with Unit 3
- **Multi-agent systems?** Begin with Unit 4, return to Unit 1-2 as reference
- **Following along with open source?** All lessons include OpenCode examples alongside Claude Code and Codex

## Certifications

This course offers two levels of certification:

### Context Fundamentals Certificate

Demonstrates you understand core context engineering concepts. Pass the Unit 1–2 quizzes with 70% or higher to earn this certificate in 2–3 weeks. It's shareable and displayed on your Hugging Face profile.

### Context Engineering Certificate

Demonstrates mastery of context engineering across all domains. Pass all Unit 1–5 quizzes (70% or higher) and complete the capstone project to earn this certificate in 5–8 weeks. It's displayed on your Hugging Face profile with a project showcase.

Both certificates verify your ability to build and maintain agent skills, connect external tools through MCPs, design multi-agent systems, and optimize context for maximum agent performance.
